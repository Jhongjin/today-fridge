# SCORE_BALANCE_V1

## 1. Balance Goal

Score should reward clean routing, not speed panic.

The first daily board uses move count only as a small mastery bonus. The main score comes from recipe completion, expiring rescue, and zero waste.

## 2. Verified Route

Route:

```text
E1, B3, C6, E5, A6, B6
```

Meaning:

1. Clear tofu using two fresh tofu and one expiring tofu.
2. Complete 김치볶음밥 using expiring rice, expiring kimchi, and expiring egg.

## 4. Interpretation

`1,700` is the first clean completion target. It should feel good but not perfect.

Future higher scores should come from:

- Extra match clears before completion.
- Combo chaining.
- Hidden item routing.
- Better remaining move efficiency up to the cap.

Do not increase pressure by lowering time. Improve mastery through route options.
