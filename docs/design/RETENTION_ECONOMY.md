# RETENTION_ECONOMY

## 1. Principles

The economy and retention design should make players return without turning the game into pressure, pay-to-win competition, or a reward-ad shell.

- Reward completion, collection, habit, and discovery more than victory.
- Use competition as motivation, not as economic payout.
- Keep Toss points separate from score, rank, win/loss, and random reward.
- Keep ads optional and placed at natural breaks.
- Keep boosters useful but non-essential.

## 2. Daily Loop

1. Open today's fridge.
2. Play one short daily board.
3. Receive completion reward.
4. Add recipe or ingredient card progress.
5. Check personal/friend/daily score.
6. Replay if score route feels improvable.

Daily rewards:

- Participation reward.
- Completion reward.
- Recipe piece or ingredient card.
- Fridge coins.
- Optional rewarded-ad bonus.

Minimum reward should exist even after failure to reduce frustration.

## 3. Weekly Loop

Weekly content should reward 3-5 days of participation, not perfect attendance.

Examples:

- Weekly ingredient bingo.
- Complete 3 different recipe categories.
- Rescue 20 expiring ingredients.
- Send 3 friend cheers.
- Finish one board without a booster.

Rewards:

- Weekly chest.
- Fridge coins.
- Recipe pieces.
- Cosmetic badge or fridge skin.

## 4. Seasonal Loop

Season length: 4 weeks.

Themes:

- Summer fridge.
- Leftover rescue week.
- Holiday fridge.
- Kimchi fridge season.
- Simple home meals.

Season rewards should be mostly collection and cosmetic:

- Seasonal recipes.
- Seasonal ingredient cards.
- Fridge skin.
- Profile frame.
- Recap card.

## 5. Currency

### Fridge Coins

Free base currency.

Earned from:

- Daily participation.
- Round completion.
- Weekly missions.
- Optional rewarded ads.
- Friend cheers.

Spent on:

- Basic boosters.
- Cosmetic fridge skins.
- Recipe hints.
- Past daily board replay.

### Recipe Pieces

Collection progression.

Earned from:

- Completing recipes.
- Daily and weekly rewards.
- Seasonal missions.

Used to:

- Unlock recipe cards.
- Fill the recipe book.
- Unlock cosmetic collection rewards.

### Fresh Tickets

Light social/event currency.

Earned from:

- Share rewards.
- Event participation.
- Friend cheer milestones.

Used for:

- Non-ranked recovery.
- Extra recipe card reveal.
- Cosmetic-only event pulls if needed.

## 6. Boosters

MVP boosters:

- `정리집게`: remove or move one selected tray ingredient.
- `냉동칸`: hold one ingredient outside the tray temporarily.
- `레시피 힌트`: highlight one useful next ingredient.

Policy:

- Boosters do not multiply score.
- Ranked daily boards should either record booster use visibly or use a no-booster leaderboard rule.
- Paid or ad-earned boosters must not create leaderboard advantage.
- Every board must be solvable without boosters.

## 7. Competition

Competition layers:

- Personal best.
- Friend challenge.
- Daily same-board leaderboard.
- Weekly small-group league.
- Seasonal honor recap.

Recommended display:

- My best score.
- Friend scores.
- Around-my-rank list.
- Top 100.
- Percentile group.

Competition rewards:

- Honor badge.
- Profile frame.
- Recap card.
- No Toss point or economic payout by rank.

## 8. Toss Point Policy

Allowed fixed conditions:

- First launch.
- Tutorial completion.
- First game completion.
- Attendance mission.
- Event page participation.
- Survey completion.

Forbidden:

- High score.
- Rank.
- Win/loss.
- Friend score victory.
- Random reward.
- Probability box.
- Booster usage.
- Difficulty-based score threshold.

Recommended wording:

- "점수와 무관하게 참여 완료 시 지급"
- "순위와 무관하게 조건 충족 시 지급"

Avoid:

- "랭킹 보상"
- "고득점 보상"
- "1등 지급"
- "상위 N명 지급"

## 9. Ads

Use rewarded ads first.

Allowed placements:

- Result screen reward bonus.
- Failure recovery.
- Daily reward extra chest.
- Shop free coin charge.

Avoid:

- During active play.
- Before result reveal.
- Before leaderboard entry.
- Ads that affect ranked score fairness.
- Ambiguous buttons that make ads look like normal reward claims.

## 10. MVP Retention Scope

Include:

- Daily board.
- Completion reward.
- Fridge coins.
- Recipe pieces.
- Today leaderboard.
- Friend challenge/share reward.
- Optional rewarded ad.
- Weekly missions.

Defer:

- Premium shop.
- Complex season pass.
- Real-time competition.
- Trading.
- Large social guilds.
- Rank-based economy.

## 11. Metrics

Core:

- D1, D3, D7 retention.
- Daily completed fridge rounds.
- Replay rate after completion.
- Leaderboard view rate.
- Friend challenge open rate.
- Recipe book revisit rate.
- Rewarded ad opt-in rate.

Risk:

- Ad view then exit rate.
- Leaderboard view then exit rate.
- Booster shortage failure loops.
- Promotion condition CS.
- Rank fairness complaints.
