# PHASE_38_BRIEF

## Queue

Personal best route memory.

## Goal

Make replay motivation more concrete by storing the ingredient route that produced a new personal best.

## Acceptance Criteria

- Best route storage is scoped to the daily board key.
- Invalid route data is ignored.
- A new clean personal best records the selected ingredient route.
- Result panel shows the best route as compact ingredient icons.
- Mobile browser tests cover the first best-route display.

## Commander Notes

This is a competition aid, not a reward. It gives players a visible path to beat without granting score, currency, or rank advantage.
