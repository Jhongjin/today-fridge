# PHASE_37_BRIEF

## Queue

MVP recipe book collection panel.

## Goal

Make recipe pieces feel collectible by giving players a compact recipe book view from the first screen.

## Acceptance Criteria

- First screen has a recipe book open action.
- Recipe book lists MVP recipes with ingredient requirements.
- Recipe book shows per-recipe piece progress.
- Active daily recipe is visually distinguished.
- Mobile browser tests cover open and close behavior.

## Commander Notes

This turns the existing recipe-piece reward into a visible collection loop without adding monetization, rank payout, or extra rules.
